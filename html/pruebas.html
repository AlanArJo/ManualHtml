ESTRUCTURA BASICA
ELEMENTO DESCRIPCION MAS INFORMACION
<!DOCTYPE> Esta declaración define el tipo de documento y la versión de HTML utilizada en la página. Por ejemplo,<!DOCTYPE html> especifica que se trata de un documento HTML5. MAS INFORMACION
<html> Este es el elemento raíz de un documento HTML. Contiene todo el contenido de la página web y define el inicio y el final del documento. MAS INFORMACION
<head> El elemento <head> se utiliza para contener metadatos y enlaces a recursos externos. Los metadatos pueden incluir información sobre el autor, la codificación de caracteres y la descripción del documento. También se utiliza para enlazar hojas de estilo y scripts. MAS INFORMACION
<meta> Este elemento se utiliza dentro del <head> para proporcionar metadatos sobre el documento, como la codificación de caracteres (<meta charset="UTF-8">), palabras clave y descripción para motores de búsqueda. MAS INFORMACION
<title> El elemento <title> se coloca dentro del <head> y define el título de la página web. Este título se muestra en la barra de título del navegador y es importante para la optimización de motores de búsqueda (SEO). MAS INFORMACION
<link> Se usa para enlazar hojas de estilo externas (CSS) al documento HTML. Esto permite separar la estructura y el contenido de la presentación y el diseño. MAS INFORMACION
<style> Este elemento se utiliza para incluir estilos CSS directamente en el documento HTML. Es útil cuando se requieren estilos específicos para una página o sección de la misma. MAS INFORMACION
<script> Se utiliza para incorporar scripts o código JavaScript en la página web. Puede ser colocado en el <head> o al final del <body> para garantizar un rendimiento óptimo. MAS INFORMACION
<base> Define la URL base para los enlaces relativos en la página. Esto es útil cuando deseas establecer una URL base común para todos los enlaces. MAS INFORMACION
<noscript> Contiene contenido que se muestra cuando el navegador no admite JavaScript o cuando JavaScript está deshabilitado. Esto permite proporcionar una experiencia básica en caso de que los scripts no se ejecuten. MAS INFORMACION
<meta charset="UTF-8"> Especifica la codificación de caracteres del documento como UTF-8, que es ampliamente utilizado y admite caracteres de diferentes idiomas. MAS INFORMACION
<body> Contiene el contenido principal de la página, incluyendo texto, imágenes y otros elementos visibles que se muestran en el navegador. MAS INFORMACION
<header> Representa la cabecera de una sección o del documento y generalmente contiene elementos como logotipos, títulos y enlaces de navegación. MAS INFORMACION
<nav> Se utiliza para definir la navegación principal de un documento, como menús o barras de navegación que permiten a los usuarios moverse por la página o el sitio web. MAS INFORMACION
<main> Define el contenido principal de un documento HTML. Debe ser único en cada página y representa la parte más importante de la misma. MAS INFORMACION
<article> Representa una sección independiente y autónoma de contenido que tiene sentido por sí misma. Por ejemplo, una publicación de blog o un artículo de noticias. MAS INFORMACION
<section> Define una sección temática dentro de un artículo o documento y generalmente se utiliza para agrupar contenido relacionado. MAS INFORMACION
<aside> Contiene contenido relacionado o complementario al contenido principal, como anuncios, barras laterales o notas adicionales. MAS INFORMACION
<footer> Representa el pie de página de una sección o del documento y generalmente incluye información de contacto o enlaces relacionados. MAS INFORMACION
<address> Define información de contacto, como una dirección de correo electrónico o una dirección física de contacto. MAS INFORMACION
<h1><h2><h3><h4><h5><h6> Estos elementos se utilizan para crear encabezados de diferentes niveles. <h1> es el encabezado más importante y suele usarse para el título principal de la página, mientras que <h6> es el menos importante en términos de jerarquía. MAS INFORMACION
<p> Define un párrafo de texto. Se utiliza para agrupar texto relacionado en párrafos distintos. MAS INFORMACION
<a> Crea hipervínculos o enlaces a otras páginas web o recursos. El atributo href especifica la URL a la que el enlace apunta. MAS INFORMACION
<abbr> Define una abreviatura y proporciona una descripción completa de la misma, que se muestra como un tooltip cuando el usuario pasa el mouse sobre ella. MAS INFORMACION
<acronym> Similar a <abbr>, pero específico para acrónimos. También proporciona una descripción completa. <strong>: Hace que el texto se muestre en negrita para resaltarlo y darle énfasis. MAS INFORMACION
<em> Hace que el texto se muestre en cursiva o enfatizado, indicando énfasis semántico. MAS INFORMACION
<mark> Resalta o marca un fragmento de texto con un color de fondo amarillo por defecto. MAS INFORMACION
<small> Reduce el tamaño del texto para indicar que es menos importante o para mostrar texto pequeño, como notas al pie. MAS INFORMACION
<sub> Coloca el texto en una posición inferior, comúnmente utilizado para escribir fórmulas químicas y matemáticas. MAS INFORMACION
<sup> Coloca el texto en una posición superior, como exponentes en ecuaciones matemáticas. MAS INFORMACION
<del> Muestra texto eliminado con una línea horizontal a través de él, típicamente usado en documentos revisados. MAS INFORMACION
<ins> Muestra texto insertado, a menudo con subrayado o una línea vertical. MAS INFORMACION
<dfn> Define una definición de un término o concepto, proporcionando claridad en el contexto. MAS INFORMACION
<time> Representa una fecha o tiempo en un formato específico, lo que facilita su interpretación por parte de las máquinas y la accesibilidad. MAS INFORMACION
<code> Se utiliza para mostrar código de programación. El texto dentro de este elemento se muestra en una fuente monoespaciada. MAS INFORMACION
<samp> Muestra una salida de programa de muestra, como resultados de comandos o código ejecutado. MAS INFORMACION
<kbd> Representa la entrada del teclado, como atajos de teclado o combinaciones de teclas. MAS INFORMACION
<pre> Preserva el formato y el espaciado del texto, útil para mostrar código o texto con formato. MAS INFORMACION
<blockquote> Define un bloque de texto cit. MAS INFORMACION
LISTAS
ELEMENTO DESCRIPCION MAS INFORMACION
<ul> Este elemento se utiliza para crear una lista desordenada en HTML. Los elementos de la lista se muestran con viñetas o puntos, y no tienen un orden específico. MÁS INFORMACIÓN
<li> El elemento <li> se utiliza dentro de <ul> o <ol> para crear elementos de lista individuales. Cada <li> representa un elemento en la lista y generalmente se muestra con viñetas o puntos. MÁS INFORMACIÓN
<ol> Este elemento se utiliza para crear una lista ordenada en HTML. En una lista ordenada, los elementos de la lista se enumeran con números, letras o números romanos, dependiendo de la configuración. MÁS INFORMACIÓN
<dl> El elemento <dl> se utiliza para crear una lista de definición en HTML. En una lista de definición, se muestra una serie de términos (definiciones) junto con sus descripciones correspondientes. MÁS INFORMACIÓN
<dt> Este elemento se utiliza dentro de <dl> para definir un término o una entrada en la lista de definición. Los términos se suelen mostrar en negrita o de alguna manera que los destaque. MÁS INFORMACIÓN
<dd> El elemento <dd> se utiliza dentro de <dl> para proporcionar la descripción o la definición del término definido en <dt>. Las descripciones suelen estar en texto normal y se muestran debajo del término correspondiente. MÁS INFORMACIÓN
TABLAS
ELEMENTO DESCRIPCION MAS INFORMACION
<table> Se utiliza para crear una tabla en HTML. Las tablas son estructuras de datos que constan de filas y columnas, organizadas en celdas. MÁS INFORMACIÓN
<caption> Este elemento se utiliza dentro de <table> para proporcionar un título o una descripción de la tabla. Aparece generalmente en la parte superior de la tabla. MÁS INFORMACIÓN
<thead> Se utiliza para definir la sección de encabezado de la tabla. Contiene filas que se utilizan para describir las columnas de la tabla. MÁS INFORMACIÓN
<tbody> Este elemento se utiliza para definir la sección de cuerpo de la tabla. Contiene filas y celdas que representan los datos reales de la tabla. MÁS INFORMACIÓN
<tfoot> Se utiliza para definir la sección de pie de tabla. Puede contener filas y celdas que se utilizan para resumir o proporcionar información adicional sobre los datos de la tabla. MÁS INFORMACIÓN
<tr> Se utiliza para crear una fila dentro de la tabla. Cada fila contiene una serie de celdas que representan datos. MÁS INFORMACIÓN
<th> Este elemento se utiliza para crear una celda de encabezado dentro de la fila de encabezado (<thead>). Se utiliza para etiquetar las columnas de la tabla y suele mostrarse en negrita y centrado. MÁS INFORMACIÓN
<td> Se utiliza para crear una celda de datos dentro de una fila en el cuerpo de la tabla (<tbody>). Contiene el contenido real de la tabla. MÁS INFORMACIÓN
FORMULARIOS
ELEMENTO DESCRIPCION MAS INFORMACION
<form> Se utiliza para crear un formulario en HTML que permite a los usuarios ingresar datos que luego se pueden enviar a un servidor para su procesamiento. Contiene elementos como campos de entrada, botones y etiquetas. MÁS INFORMACIÓN
<input> <input> se utiliza para crear campos de entrada en un formulario, como campos de texto, casillas de verificación, botones de radio, campos ocultos, etc. MÁS INFORMACIÓN
<textarea> <textarea> se utiliza para crear áreas de texto multilínea en un formulario, donde los usuarios pueden ingresar texto más largo, como comentarios o mensajes largos. MÁS INFORMACIÓN
<select> <select> se utiliza para crear una lista desplegable en un formulario. Puede contener múltiples opciones representadas por elementos <option>. MÁS INFORMACIÓN
<optgroup> <optgroup> se utiliza dentro de <select> para agrupar opciones relacionadas en la lista desplegable. Ayuda a organizar y estructurar las opciones. MÁS INFORMACIÓN
<option> <option> se utiliza dentro de <select> para definir las opciones disponibles en la lista desplegable. MÁS INFORMACIÓN
<button> <button> se utiliza para crear botones en un formulario, que los usuarios pueden hacer clic para enviar el formulario o realizar otras acciones especificadas. MÁS INFORMACIÓN
<label> <label> se utiliza para proporcionar una etiqueta descriptiva para un campo de entrada. Ayuda a los usuarios a comprender el propósito del campo y mejora la accesibilidad. MÁS INFORMACIÓN
<fieldset> <fieldset> se utiliza para agrupar un conjunto de campos relacionados en un formulario. Puede contener múltiples campos y suele ir acompañado de una etiqueta <legend> para describir el conjunto. MÁS INFORMACIÓN
<legend> <legend> se utiliza junto con <fieldset> para proporcionar una descripción o título para un conjunto de campos en un formulario. MÁS INFORMACIÓN
<datalist> <datalist> se utiliza junto con campos de entrada <input> para proporcionar una lista de sugerencias al usuario a medida que escribe en el campo. MÁS INFORMACIÓN
<output> <output> se utiliza para mostrar resultados o cálculos basados en la entrada del usuario dentro del formulario. MÁS INFORMACIÓN
<meter> <meter> se utiliza para crear medidores de rango, como barras de progreso o indicadores de nivel, para representar valores numéricos dentro de un rango específico. MÁS INFORMACIÓN
<progress> <progress> se utiliza para crear barras de progreso que indican el progreso de una tarea en un formulario, como la carga de un archivo o el envío de datos. MÁS INFORMACIÓN
MULTIMEDIA
ELEMENTO DESCRIPCION MAS INFORMACION
<img> <img> se utiliza para incrustar imágenes en una página web. Puedes especificar la fuente de la imagen utilizando el atributo <src> y proporcionar texto alternativo para la accesibilidad con el atributo <alt>. MÁS INFORMACIÓN
<audio> <audio> se utiliza para agregar archivos de audio a una página web. Puedes especificar la fuente del archivo de audio utilizando el atributo <src> y proporcionar controles de reproducción con atributos como <controls>. MÁS INFORMACIÓN
<video> <video> se utiliza para agregar archivos de video a una página web. Al igual que con <audio>, puedes especificar la fuente del archivo de video y proporcionar controles de reproducción. MÁS INFORMACIÓN
<source> <source> se utiliza dentro de elementos <audio> y <video> para especificar múltiples fuentes de medios. Esto permite al navegador seleccionar la mejor fuente de medios según el tipo y las capacidades del navegador. MÁS INFORMACIÓN
<track> <track> se utiliza para agregar pistas de subtítulos o descripciones de audio a elementos de medios como <video> y <audio>. Puedes definir el tipo de pista y la fuente de la misma. MÁS INFORMACIÓN
<map> <map> se utiliza junto con elementos <area> para crear mapas de imágenes interactivos. Puedes definir áreas en una imagen y vincularlas a enlaces o acciones específicas en la página web. MÁS INFORMACIÓN
<area> <area> se utiliza dentro de un elemento <map> y define áreas en una imagen que se pueden hacer clic o activar de alguna otra manera. Cada área se vincula a una URL o acción específica. MÁS INFORMACIÓN
INCRUSTACION
ELEMENTO DESCRIPCION MAS INFORMACION
<iframe> <iframe> se utiliza para incrustar contenido externo dentro de una página web, como otra página web, videos de YouTube, mapas de Google, etc. Puedes especificar la fuente del contenido utilizando el atributo <src>. MÁS INFORMACIÓN
<object> <object> se utiliza para incrustar objetos multimedia, como Flash, en una página web. Puedes especificar la fuente y el tipo del objeto, así como proporcionar contenido alternativo en caso de que el navegador no pueda mostrar el objeto. MÁS INFORMACIÓN
<embed> <embed> se utiliza para incrustar contenido multimedia, como audio o video, en una página web. Debes especificar la fuente y el tipo del contenido, así como las dimensiones de la visualización. MÁS INFORMACIÓN
<canvas> <canvas> se utiliza para dibujar gráficos, imágenes y animaciones en una página web utilizando JavaScript. Proporciona un lienzo en blanco en el que puedes dibujar dinámicamente. MÁS INFORMACIÓN
<svg> <svg> se utiliza para crear gráficos vectoriales escalables (SVG) en una página web. Los gráficos SVG son imágenes vectoriales que se pueden escalar sin pérdida de calidad y se pueden animar y manipular con JavaScript. MÁS INFORMACIÓN
CONTENIDO TEXTUAL
ELEMENTO DESCRIPCION MAS INFORMACION
<mark> <mark> se utiliza para resaltar o marcar partes de un texto dentro de un párrafo u otro elemento de texto. Es útil para resaltar términos de búsqueda o partes importantes del contenido. MÁS INFORMACIÓN
<ruby>, <rt>, <rp> <ruby>, <rt> y <rp> se utilizan para representar texto ruby, que es una forma de anotación de texto comúnmente utilizada en la tipografía asiática. <ruby> contiene el texto base, <rt> contiene la anotación, y <rp> proporciona una opción de representación en caso de que el navegador no admita ruby. MÁS INFORMACIÓN
<bdo> <bdo> se utiliza para especificar la dirección de texto en la que debe mostrarse el contenido. Puede usarse para forzar el texto a mostrarse de derecha a izquierda o de izquierda a derecha, independientemente de la dirección del idioma de la página. MÁS INFORMACIÓN
<wbr> <wbr> se utiliza para sugerir un lugar potencial de división de una palabra larga en una línea de texto. Es útil para mejorar la legibilidad y la apariencia de páginas web con texto largo y evita que las palabras se corten de manera poco estética en líneas separadas. MÁS INFORMACIÓN
<ins> <ins> se utiliza para marcar texto que ha sido insertado en el documento, por ejemplo, en comparación con una versión anterior del mismo documento. Puede ser útil para resaltar adiciones o cambios en el contenido. MÁS INFORMACIÓN
<del> <del> se utiliza para marcar texto que ha sido eliminado o tachado en el documento, por ejemplo, en comparación con una versión anterior del mismo documento. Puede ser útil para mostrar supresiones o cambios en el contenido. MÁS INFORMACIÓN
<blockquote> <blockquote> se utiliza para definir un bloque de texto citado en HTML. Suelen representar fragmentos de texto de otras fuentes y se formatean de manera especial, como con sangría o un estilo diferente, para indicar que son citas textuales. MÁS INFORMACIÓN
AGRUPACION DE CONTENIDO
ELEMENTO DESCRIPCION MAS INFORMACION
<header> <header> se utiliza para representar la sección de encabezado de una página web o de una sección de contenido. Generalmente contiene elementos como logotipos, títulos y enlaces de navegación que son relevantes para la parte superior de la página o sección. MÁS INFORMACIÓN
<nav> <nav> se utiliza para definir una sección de navegación en una página web. Contiene enlaces de navegación que permiten a los usuarios acceder a diferentes partes del sitio o a otras páginas relacionadas. MÁS INFORMACIÓN
<footer> <footer> se utiliza para representar la sección de pie de página de una página web o de una sección de contenido. Generalmente contiene información de contacto, enlaces a políticas de privacidad y otros elementos relevantes para el final de la página o sección. MÁS INFORMACIÓN
<main> <main> se utiliza para representar el contenido principal de una página web. Debe ser único en cada página y no debe usarse para encabezados, pie de página o navegación. Ayuda a mejorar la accesibilidad y la estructura de la página. MÁS INFORMACIÓN
CONTENIDO SEMANTICO
ELEMENTO DESCRIPCION MAS INFORMACION
<article> <article> se utiliza para representar una pieza de contenido independiente y autónoma en una página web. Debe tener sentido por sí mismo y ser reutilizable en otros contextos, como una entrada de blog, una noticia o un artículo. MÁS INFORMACIÓN
<section> <section> se utiliza para agrupar contenido relacionado en una página web. Ayuda a dividir y estructurar el contenido de manera semántica, como capítulos en un libro o secciones en un artículo. MÁS INFORMACIÓN
<aside> <aside> se utiliza para representar contenido relacionado pero no esencial en una página web. Puede contener información como barras laterales, publicidad, notas relacionadas y más. A menudo se coloca junto al contenido principal pero no es parte fundamental de este. MÁS INFORMACIÓN
<details> <details> se utiliza para crear un widget de detalles en una página web que permite al usuario mostrar u ocultar contenido adicional. El contenido se coloca dentro de <details>, y su título se define con el elemento <summary>. MÁS INFORMACIÓN
<summary> <summary> se utiliza en combinación con el elemento <details> para proporcionar un título o encabezado que el usuario puede hacer clic para mostrar u ocultar el contenido adicional dentro del widget de detalles. MÁS INFORMACIÓN
<figure> <figure> se utiliza para encapsular contenido multimedia, como imágenes o gráficos, junto con su descripción utilizando el elemento <figcaption>. Ayuda a proporcionar una estructura semántica a los contenidos relacionados con multimedia. MÁS INFORMACIÓN
<figcaption> <figcaption> se utiliza junto con el elemento <figure> para proporcionar una descripción o título para el contenido multimedia que está encapsulado en <figure>. Ayuda a mejorar la accesibilidad y la comprensión del contenido. MÁS INFORMACIÓN
ELEMENTOS DE LINEA Y TEXTO
ELEMENTO DESCRIPCION MAS INFORMACION
<a> <a> se utiliza para crear hipervínculos o enlaces en una página web. Puedes especificar la URL de destino en el atributo <href> y el texto del enlace entre las etiquetas <a>. MÁS INFORMACIÓN
<abbr> <abbr> se utiliza para definir una abreviatura o acrónimo en el texto. Puedes proporcionar una descripción completa utilizando el atributo <title> para mejorar la accesibilidad. MÁS INFORMACIÓN
<b> <b> se utiliza para aplicar un estilo de texto en negrita al contenido. Sin embargo, no tiene un significado semántico específico y se recomienda utilizar <strong> en su lugar para textos importantes. MÁS INFORMACIÓN
<br> <br> se utiliza para crear un salto de línea en el texto. No necesita una etiqueta de cierre y se usa para separar el contenido en diferentes líneas o para introducir espacios en blanco en el texto. MÁS INFORMACIÓN
<cite> <cite> se utiliza para marcar el título de una obra, como un libro o una película, en el texto. Proporciona un formato especial para la cita y se utiliza para indicar la fuente de una referencia. MÁS INFORMACIÓN
<code> <code> se utiliza para representar fragmentos de código en el texto. Mantiene el formato de fuente monoespaciada y se usa para destacar código o comandos en el contenido web. MÁS INFORMACIÓN
<em> <em> se utiliza para enfatizar el texto y se representa generalmente en cursiva. Puede indicar que una palabra o frase es importante en el contexto del contenido y es un elemento de estilo en lugar de formato estricto de presentación. MÁS INFORMACIÓN
<i> <i> se utiliza para aplicar estilo en cursiva al texto. Al igual que <b>, no tiene un significado semántico específico y se recomienda utilizar <em> en su lugar para énfasis en el contenido. MÁS INFORMACIÓN
<mark> <mark> se utiliza para resaltar o marcar partes de un texto dentro de un párrafo u otro elemento de texto. Es útil para resaltar términos de búsqueda o partes importantes del contenido. MÁS INFORMACIÓN
<q> <q> se utiliza para marcar citas cortas o textos que deben aparecer entre comillas en el contenido. Agrega comillas automáticamente al texto y se utiliza para indicar que el texto es una cita directa de otra fuente. MÁS INFORMACIÓN
<s> <s> se utiliza para representar texto que ha sido tachado o eliminado del contenido. A menudo se usa en revisiones o cambios en el contenido para mostrar lo que se ha eliminado. No se debe utilizar para agregar énfasis o importancia negativa al texto. MÁS INFORMACIÓN
<small> <small> se utiliza para reducir el tamaño del texto en relación con el texto circundante. Puede ser útil para notas al pie o texto legal que debe aparecer en un tamaño de fuente más pequeño. MÁS INFORMACIÓN
<span> <span> se utiliza como un contenedor genérico para aplicar estilos o scripts a partes específicas del contenido de texto. No tiene un significado semántico propio, pero se puede utilizar junto con CSS y JavaScript para manipular el contenido de manera más precisa. MÁS INFORMACIÓN
<strong> <strong> se utiliza para aplicar un estilo de texto en negrita y para indicar que el texto contenido es de importancia fuerte o enfática en el contexto. Se recomienda su uso en lugar de <b> para enfatizar el significado semántico del texto. MÁS INFORMACIÓN
<sub> <sub> se utiliza para representar texto subíndice, que es texto que aparece más pequeño y ligeramente por debajo de la línea de base del texto normal. Se usa comúnmente en fórmulas químicas o matemáticas. MÁS INFORMACIÓN
<sup> <sup> se utiliza para representar texto superíndice, que es texto que aparece más pequeño y ligeramente por encima de la línea de base del texto normal. Se usa comúnmente en fórmulas químicas o matemáticas, así como en notas al pie o exponentes. MÁS INFORMACIÓN
ELEMENTOS DE CONTENIDO INTERACTIVO
ELEMENTO DESCRIPCION MAS INFORMACION
<details> <details> se utiliza para crear un widget de detalles en una página web que permite al usuario mostrar u ocultar contenido adicional. El contenido se coloca dentro de <details>, y su título se define con el elemento <summary>. MÁS INFORMACIÓN
<summary> <summary> se utiliza en combinación con el elemento <details> para proporcionar un título o encabezado que el usuario puede hacer clic para mostrar u ocultar el contenido adicional dentro del widget de detalles. MÁS INFORMACIÓN
<menu> <menu> se utiliza para definir un menú contextual o de aplicaciones en una página web. Puede contener elementos <menuitem> que representan las opciones del menú y se utilizan en aplicaciones web para acciones contextuales. MÁS INFORMACIÓN
<menuitem> <menuitem> se utiliza dentro de un elemento <menu> para representar una opción de menú en un menú contextual o de aplicaciones. Cada <menuitem> puede tener un comando asociado que se ejecuta cuando se selecciona la opción del menú. MÁS INFORMACIÓN
<dialog> <dialog> se utiliza para crear cuadros de diálogo modales en una página web. Los cuadros de diálogo modales son ventanas emergentes que requieren la interacción del usuario antes de continuar con la navegación. <dialog> contiene contenido adicional que se muestra en el cuadro de diálogo. <figure> <figure> se utiliza para encapsular contenido multimedia, como imágenes o gráficos, junto con su descripción utilizando el elemento <figcaption>. Ayuda a proporcionar una estructura semántica a los contenidos relacionados con multimedia. MÁS INFORMACIÓN
<figcaption> <figcaption> se utiliza junto con el elemento <figure> para proporcionar una descripción o título para el contenido multimedia que está encapsulado en <figure>. Ayuda a mejorar la accesibilidad y la comprensión del contenido. MÁS INFORMACIÓN

<ins> <ins> se utiliza para representar texto o contenido que ha sido insertado en un documento o página web. Es útil para mostrar adiciones o cambios en el contenido y se suele mostrar subrayado por defecto en la mayoría de los navegadores. MÁS INFORMACIÓN
<del> <del> se utiliza para representar texto o contenido que ha sido eliminado o tachado de un documento o página web. Es útil para mostrar eliminaciones o cambios en el contenido y se suele mostrar con una línea a través del texto en la mayoría de los navegadores. MÁS INFORMACIÓN
<s> <s> se utiliza para representar texto o contenido que ha sido tachado o eliminado del contenido original. Es similar a <del> y se utiliza para mostrar cambios o eliminaciones, pero no tiene el mismo significado semántico en términos de tiempo o contexto que <del> o <ins>. MÁS INFORMACIÓN
<strike> <strike> se solía utilizar para representar texto tachado, similar a <s> y <del>. Sin embargo, ha sido obsoleto en HTML5, y se recomienda usar <del> o <s> en su lugar para representar eliminaciones o cambios en el contenido. MÁS INFORMACIÓN
<abbr> <abbr> se utiliza para definir una abreviatura o acrónimo en el texto. Puedes proporcionar una descripción completa utilizando el atributo <title> para mejorar la accesibilidad y explicar el significado de la abreviatura o acrónimo. MÁS INFORMACIÓN
<dfn> <dfn> se utiliza para representar la definición de un término en el texto. Es útil cuando se desea marcar un término específico en el contenido y proporcionar una explicación o definición más detallada utilizando el atributo <title>. MÁS INFORMACIÓN
<cite> <cite> se utiliza para marcar el título de una obra, como un libro o una película, en el texto. Proporciona un formato especial para la cita y se utiliza para indicar la fuente de una referencia o trabajo citado en el contenido. MÁS INFORMACIÓN
<blockquote> <blockquote> se utiliza para definir un bloque de texto citado en el contenido. Por lo general, se usa para citas largas o bloques de texto de otros autores y ayuda a distinguir visualmente el texto citado del texto circundante. MÁS INFORMACIÓN
<q> <q> se utiliza para marcar citas cortas o textos que deben aparecer entre comillas en el contenido. Agrega comillas automáticamente al texto y se utiliza para indicar que el texto es una cita directa de otra fuente o una frase corta que se debe destacar. MÁS INFORMACIÓN
<address> <address> se utiliza para representar información de contacto o detalles de dirección en el contenido. A menudo se coloca en el pie de página de una página web y puede incluir información como direcciones de correo electrónico o números de teléfono de contacto. MÁS INFORMACIÓN

<hr> <hr> se utiliza para crear una línea horizontal en una página web. Generalmente se utiliza para separar secciones de contenido o elementos visuales en la página y no requiere una etiqueta de cierre. MÁS INFORMACIÓN
<pre> <pre> se utiliza para representar texto preformateado, lo que significa que el texto se muestra exactamente como está escrito en el código fuente, manteniendo espacios en blanco, saltos de línea y tabulaciones. Es útil para mostrar código o texto con formato especial donde la presentación exacta es importante. MÁS INFORMACIÓN

<b> <b> se utiliza para aplicar formato de texto en negrita a un elemento o parte del texto. Sin embargo, no tiene un significado semántico específico y se recomienda su uso limitado en favor de <strong> para enfatizar el significado semántico del texto. MÁS INFORMACIÓN
<em> <em> se utiliza para aplicar formato de texto en cursiva o énfasis a un elemento o parte del texto. Se utiliza para resaltar palabras o frases que deben enfatizarse en el contexto del contenido. MÁS INFORMACIÓN
<i> <i> se utiliza para aplicar formato de texto en cursiva a un elemento o parte del texto. Sin embargo, al igual que <b>, no tiene un significado semántico específico y se recomienda su uso limitado en favor de <em> para enfatizar el significado semántico del texto en itálica. MÁS INFORMACIÓN
<strong> <strong> se utiliza para aplicar formato de texto en negrita y para indicar que el texto contenido es de importancia fuerte o enfática en el contexto. Se recomienda su uso en lugar de <b> para enfatizar el significado semántico del texto. MÁS INFORMACIÓN
<small> <small> se utiliza para representar texto o contenido que es más pequeño en tamaño que el texto circundante. Puede ser útil para notas al pie, leyendas o contenido que requiera un tamaño de fuente más pequeño para fines estilísticos o de diseño. MÁS INFORMACIÓN
<sub> <sub> se utiliza para representar texto subíndice, que es texto que aparece más pequeño y ligeramente por debajo de la línea de base del texto normal. Se usa comúnmente en fórmulas químicas o matemáticas para representar índices inferiores o en notaciones científicas. MÁS INFORMACIÓN
<sup> <sup> se utiliza para representar texto superíndice, que es texto que aparece más pequeño y ligeramente por encima de la línea de base del texto normal. Se usa comúnmente en fórmulas químicas o matemáticas, así como en notas al pie o exponentes en textos científicos o técnicos. MÁS INFORMACIÓN
<mark> <mark> se utiliza para resaltar partes del texto o del contenido con un color de fondo amarillo. Es útil para señalar términos de búsqueda o partes importantes del contenido que se deseen destacar visualmente en una página web. MÁS INFORMACIÓN
<ruby>, <rt>, <rp> <ruby> se utiliza para representar texto ruby, que es una anotación o explicación pequeña que se muestra junto a un carácter o palabra. <rt> se utiliza para definir el texto ruby y <rp> se utiliza para proporcionar una alternativa de visualización en navegadores que no admiten ruby. Es común en idiomas como el japonés para la lectura de caracteres kanji. MÁS INFORMACIÓN
<bdo> <bdo> se utiliza para especificar la dirección del texto en un elemento, como de derecha a izquierda para idiomas como el árabe o de izquierda a derecha para la mayoría de los idiomas occidentales. Puede ser útil cuando se necesita controlar la dirección del texto en elementos específicos. MÁS INFORMACIÓN
